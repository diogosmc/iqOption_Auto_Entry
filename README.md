🤖 IQ Option - Scheduled Trading Bot
Este projeto é uma ferramenta de automação desenvolvida em Python para a plataforma IQ Option. Ele permite o agendamento de operações em pares de Opções Digitais, executando entradas precisas em horários pré-configurados pelo usuário.

Nota: Este projeto foi desenvolvido de forma legada, servindo como registro da minha evolução em lógica de programação e integração de APIs financeiras.

🚀 Funcionalidades
Agendamento Preciso: Realiza operações automaticamente no horário definido.

Foco em Opções Digitais: Otimizado para execução em pares de moedas no formato digital.

Configuração Dinâmica: Gerenciamento de credenciais e parâmetros via arquivo externo (INFOS.txt).

🛠️ Tecnologias Utilizadas
Linguagem: Python

Integração: API IQ Option (Biblioteca de conexão via WebSocket)

Gestão de Dados: Manipulação de arquivos de texto para configuração de parâmetros.

📦 Como Utilizar
Pré-requisitos: Certifique-se de ter o Python instalado em sua máquina.

Configuração: Abra o arquivo INFOS.txt e insira suas credenciais e os horários das operações seguindo o padrão indicado.

Execução:

Bash
python main.py
